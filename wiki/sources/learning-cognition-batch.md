## Summary by Source

### 1. 合意困难 (Desirable Difficulties)

Robert Bjork 的核心理论。学习条件在短期内感觉更难、更慢，但产生显著更持久和可迁移的长期学习。关键区分：**Performance ≠ Learning**。新遗忘理论将记忆分为 Storage Strength（永不衰减）和 Retrieval Strength（波动），困难检索建立更强存储。

→ [[合意困难-Desirable-Difficulties]]

### 2. 渐进摘要 (Progressive Summarization)

Tiago Forte 的五层笔记提炼法。从原始捕获 → 加粗 → 高亮 → 自写摘要 → 跨笔记连接，逐层提炼。解决"笔记太丰富→检索困难"的矛盾。关键原则：第 4-5 层必须人工完成，AI 可协助 1-3 层。

→ [[渐进摘要-Progressive-Summarization]]

### 3. 知识库的正确打开方式 (KB Correct Usage Pattern)

知识库本身是中性的。问题在于**使用方式**：把它当记忆义肢而非验证工具。正确顺序：闭卷回忆→写下答案→打开 KB 对比→修正差异。三层次认知分工：大脑做检索/连接/综合，自写摘要做元认知监控触发器，AI+KB 做存储和候选提示。

→ [[知识库的正确打开方式-KB-Correct-Usage-Pattern]]

### 4. 认知卸载 (Cognitive Offloading)

使用外部工具减轻认知负荷的三层分类：辅助型（低风险）、替代型（中风险）、破坏型（高风险）。"Google 效应"：人们记住信息存储位置优于信息本身。关键区分：卸载**存储**有益，卸载**思考**有害。

→ [[认知卸载-Cognitive-Offloading]]
